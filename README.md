# Maki-Cafe-Co-Working-Space-web
Project web sederhana memenuhi tugas mata kuliah front-end web development

## Deskripsi Singkat
Website ini dibuat untuk memperkenalkan MAKI Cafe & Co-Working Space kepada pengunjung.  
Fitur utama meliputi informasi menu, fasilitas co-working, galeri foto, serta sistem reservasi online.  
Tujuan project adalah memberikan pengalaman digital yang informatif dan responsif bagi pengguna.

## Cara Menjalankan Website
1. Buka repository di GitHub.
2. Pastikan branch main sudah berisi file project.
3. Aktifkan GitHub Pages:
   - Masuk ke *Settings* → *Pages*.
   - Pilih branch main dan folder /root.
   - Simpan pengaturan.
4. Website akan bisa diakses melalui link GitHub Pages.
